# Docs Framework

Build beautiful documentation sites with **Next.js + MDX**, powered by a flexible content system.

---

## ✨ Why this exists

Most documentation tools are:

* Too complex
* Too opinionated
* Hard to customize

Docs Framework is different:

* 🧩 **Simple architecture**
* ⚡ **Static export friendly**
* 🔌 **Pluggable content sources (local / GitHub)**
* 🎨 **Fully customizable UI**

---

## 🚀 What you get

* File-based MDX docs
* Auto-generated sidebar
* Breadcrumb navigation
* Previous / Next navigation
* Folder-based organization
* SEO-ready pages

---

## 📁 Example Structure

```txt
content/docs/
├── introduction.mdx
├── getting-started.mdx
├── guides/
│   ├── intro.mdx
│   └── advanced/
│       └── auth.mdx
```

---

## 🧠 How it works

1. You write docs in **MDX**
2. Framework builds a **docs tree**
3. Routes are generated automatically
4. Pages are rendered statically
